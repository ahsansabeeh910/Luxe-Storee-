## 🛍 About

LUXE Store is a premium full-stack eCommerce platform designed for luxury shopping experiences. The application provides secure authentication, product browsing, dynamic product management, shopping cart functionality, wishlist support, order management, and responsive UI components powered by a modern backend architecture.

---

## 🚀 Features

✔ User Authentication (Login / Signup)
✔ JWT Protected Routes
✔ Wishlist System ❤️
✔ Shopping Cart 🛒
✔ Product Categories & Filtering
✔ Search Functionality 🔍
✔ Dynamic Products from MongoDB
✔ Product Detail Modal
✔ Order Placement System
✔ Product Images Support
✔ Responsive UI
✔ MongoDB Integration
✔ Secure Backend APIs

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/ahsansabeeh910/Luxe-Storee-.git
cd luxe-store
```

### Install Backend Dependencies

```bash
cd backend
npm install
```

### Run Backend Server

```bash
node server.js
```

### Open Frontend

```text
frontend/ecommerce_store.html
```

Use Live Server or open directly in browser.

---

## 🔗 API Routes

### Products

```text
GET /api/products
```

### Cart

```text
GET /api/cart
POST /api/cart
DELETE /api/cart/:id
```

### Orders

```text
POST /api/orders
GET /api/orders/myorders
```

### Authentication

```text
POST /api/auth/signup
POST /api/auth/login
```
